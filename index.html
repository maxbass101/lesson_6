overflow — управляє відображенням контенту блочного елементу.

Не успадковується.

  • visible — відображається усесь контент елемента, навіть за межами встановлених розмірів (за замовченням);
  • hidden — відображається тільки контент всередені елемента, все інше буде сховано;
  • scroll — завжди додаються смужки прокрутки;
  • auto — смужки прокрутки додаються лише за необхідністю.

display

display — багатоцільова властивість яка задає глобальні параметри тега.

Не успадковується.

  • block — елемент відображається як блочний (за замовченням для блочних об'ектів);
  • inline — елемент відображається як рядковий (за замовченням для рядкових об'ектів);
  • inline-block — елемент відображається як рядковий але з можливістю використання усіх CSS-властивостей (блочно-рядковий);
  • none — елемент не виводиться в браузер. Інші елементи займають його місце. Не приймає участь у обробці браузером.
